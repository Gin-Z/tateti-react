Notas sobre este proyecto
Este ta-te-ti fue una de mis primeras prácticas con React.
Me ayudó a entender:
- manejo de estado
- renderizado condicional
- props y componentes simples

Luego desarrollé una versión más completa y mejorada (4 en línea), aplicando lo aprendido y agregando:
- mejores prácticas
- lógica más compleja
- componentes reutilizables
- un diseño más cuidado

Podés ver la versión mejorada aquí: https://github.com/Gin-Z/4-en-linea-react-js
